HTML Tables

What is an HTML Table?

An HTML Table organizes data in rows and columns in a tabular format. Tables are great for presenting data and creating quick references between different types of information. HTML tables are also foundational in many layouts, although CSS grids and flexboxes are more commonly used today.

Defining Tables in HTML

An HTML table is defined with the <table> tag. Each row is defined with <tr>, table headers with <th>, and cells with <td>.

Table Syntax


<!-- index.html -->
<!DOCTYPE html>
<html>
<head>
<style>
    table, th, td { border: 1px solid black; }
</style>
</head>
<body>
    <table>
        <tr><th>Book Name</th><th>Author</th><th>Genre</th></tr>
        <tr><td>The Book Thief</td><td>Markus Zusak</td><td>Historical</td></tr>
        <tr><td>The Cruel Prince</td><td>Holly Black</td><td>Fantasy</td></tr>
    </table>
</body>
</html>

Common Table Tags

Tag Description
<table> Defines the structure for organizing data.
<tr> Represents a row in the table.
<th> Defines a table header cell, typically bold and centered.
<td> Defines a table data cell.
<caption> Provides a title or description for the table.
